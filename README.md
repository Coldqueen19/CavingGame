# Cave Crawler: The Ultimate Text Adventure
**The largest text-based RPG I have ever developed!**

Welcome to a complicated journey through a massive network of levels (caves). This isn't just a simple text game; it's a well built world featuring equipment mechanics, ability improving items, and a level progression system.

## Key Features
*   **Intensive Exploration:** Traverse an enormous world filled with a variety of unique caves, each one expertly crafted with challenges and varying story paths.
*   **Simple, Easy to use Item System:** Find and equip gear that provides **permanent** boosts to your in game abilities.
*   **Progress Tracking:** Features an automatic save system that tracks your journey through the levels with no player intervention.

## Save Mechanics (How it Works)
To make sure that the difficulty and "rogue-lite" feel of the game is always present, the save system uses a **Levels unlocked system**:
*   **What is saved:** The game automatically every new **Level** you unlock
*   **Startup:** When you start the game, you will begin at the **main menu** of the game or in a specialised menu for specific sections. Note that progress *inside* a specific cave (your exact choice) is not saved—only the caves unlocked.

## How to Play
1.  **Launch:** Run the Cave Crawler executable.
2.  **Navigate:** Use the text prompts to explore the levels.
If you wish to start fresh, simply choose the reset option in the main menu(this only becomes available after the first 2 caves, with some exceptions)

## Installation & Setup
